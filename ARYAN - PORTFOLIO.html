★ Project 01

Human Action Detection

A real-time computer vision system that watches a live camera feed and classifies what a person is doing — no dataset upload, no latency. Point it at anyone and it instantly reads body language, movement, and posture.

  • Detects Walking — tracks stride and velocity patterns from pose keypoints
  • Recognises Hand Waving — monitors wrist trajectory oscillation over frames
  • Identifies Dancing — rapid multi-joint movement classification
  • Classifies Standing / Sitting — hip-to-knee angle threshold logic
  • Live bounding box overlay with action label rendered on-screen in real time

Diet Plan Generator

A personalised nutrition calculator that takes 6 user inputs and outputs precise macro targets, hydration goals, and smart fitness tips — then lets the user download their full plan as a formatted PDF.

  • Inputs: Body weight, goal (fat loss / maintenance / muscle gain)
  • Workout frequency per week + meal type (veg / vegan / non-veg)
  • Number of meals per day (3, 4 or 5) + any existing health conditions
  • Outputs protein grams, carb grams, and daily water intake targets
  • Bonus tips per goal — e.g. "45 min cardio daily" for fat loss users
  • One-click PDF download of the complete personalised plan
